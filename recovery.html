Manual Recovery Instructions

If the above doesn't work, try these manual steps:

  1. Chrome Extension Method:
    • Go to chrome://extensions/
    • Find "My Start Page" extension
    • Click "Inspect views: service worker" or "background page"
    • In the console, type: chrome.storage.local.get(null, console.log)
    • Look for "customLinksWithCategories" in the output
  2. Browser Storage Method:
    • Open any webpage and press F12
    • Go to "Application" tab â†’ "Local Storage"
    • Look for entries starting with "customLinks"
  3. Extension Recovery URL:
    • Try: chrome-extension://[extension-id]/recovery.html
    • Replace [extension-id] with your actual extension ID
